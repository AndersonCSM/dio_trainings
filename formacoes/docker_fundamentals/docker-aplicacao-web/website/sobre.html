Sobre o Projeto

Este é um projeto simples focado em testar as funcionalidades do Docker Compose utilizando um servidor Apache para hospedar uma aplicação web estática.

A proposta é simular um pequeno site institucional, utilizando Bootstrap para a parte visual e containers Docker para facilitar o deploy e a portabilidade da aplicação.
